Visual Shop

A minecraft plugin for selling items, with a dropped item as representation

Libraries

The plugin needs the MySQL JDBC library, whether in the libs folder of craftbukkit, or in your environnement path. You can download it from MySQL website, and choose "Platform independant".

Downloads

 Plugin's jar files. Prefer to use the latest stable build than the latest beta, which could have bugs. To see all the other version, changelog included, click on "Download center".
